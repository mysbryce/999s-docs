## Debug - ดีบั๊ก

เรามีฟังก์ชั่นสำหรับการทำ Debug Mode พร้อมเรียบร้อยแล้ว เพื่อให้คุณสามารถตรวจสอบและหาจุดผิดพลาดได้ง่ายมากยิ่งขึ้น

### Enable / Disable Debug Mode - การเปิดปิดโหมด

```lua
-- NS.setDebugMode(enabled)

NS.setDebugMode(true)
```

| Parameter | Type    | Description                      | Note |
| --------- | ------- | -------------------------------- | ---- |
| enabled   | boolean | คุณจะเปิดการใช้งาน debug หรือไม่ |      |

### Debug To Console - การแสดงผลดีบั๊ก

```lua
-- NS.debug(type, message)

NS.debug('info', 'Test info')
```

| Parameter | Type      | Description | Note |
| --------- | --------- | ----------- | ---- |
| type      | TypesList |             |      |
| message   | string    |             |      |

### Types List

1. success
2. info
3. warn
4. error

ระบบสำหรับการ Debug ข้อมูลต่าง ๆ เพื่อให้สามารถแยกหรือระบุรูปแบบได้ชัดเจน

## Toggle Mode

<CodeGroup>
```lua client/your-client-file.lua
self:setDebugMode(true)
```
</CodeGroup>

### Paramaters

<ParamField path="enabled" type="boolean" required>
    เปิดใช้งาน Debug หรือไม่
</ParamField>

## Trace Debug Message

<CodeGroup>
```lua client/your-client-file.lua
self:debug('success', 'Success Debug')
self:debug('info', 'Info Debug')
self:debug('warn', 'Warn Debug')
self:debug('error', 'Error Debug')
```
</CodeGroup>

### Paramaters

<ParamField path="prefix" type="enum" required>
    <Expandable title="enums">
        - `success`
        - `info`
        - `warn`
        - `error`
    </Expandable>
</ParamField>

<ParamField path="text" type="string" required>
    ข้อความที่จะนำไปแสดงผล
</ParamField>

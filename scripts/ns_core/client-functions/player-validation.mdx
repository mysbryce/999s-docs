ระบบสำหรับการตรวจสอบข้อมูลของผู้เล่นที่จะทำให้คุณตรวจสอบข้อมูลได้ง่ายขึ้น

## Item Validation

<CodeGroup>
```lua client/your-client-file.lua
local item = self:getItem('bread', false)
print(ESX.DumpTable(item))
```
</CodeGroup>

### Parameters

<ParamField path="name" type="string" required>
  ชื่อไอเทม
</ParamField>
<ParamField path="validate" type="boolean">
  ตรวจสอบบน Server หรือไม่ ถ้าหากไม่กำหนดหรือ false จะตรวจสอบเฉพาะ Client
</ParamField>

### Return

<ParamField path="ESXItem?" type="{ name: string, label: string, count: integer, weight: number, limit: integer, rare: boolean, canRemove: boolean } | nil" required>
  รายการข้อมูลไอเทม
</ParamField>

## Account Validation

<CodeGroup>
```lua client/your-client-file.lua
local account = self:getAccount('money', false)
print(ESX.DumpTable(account))
```
</CodeGroup>

### Parameters

<ParamField path="name" type="string" required>
  ชื่อบัญชี
</ParamField>
<ParamField path="validate" type="boolean">

</ParamField>

### Return

<ParamField path="ESXAccount?" type="{ name: string, label: string, money: number } | nil" required>
  รายการข้อมูลบัญชี
</ParamField>

ระบบสำหรับการตรวจสอบข้อมูลของผู้เล่นที่จะทำให้คุณตรวจสอบข้อมูลได้ง่ายขึ้น

## Item Validation

<CodeGroup>
```lua client/your-client-file.lua
local item = self:getItem('bread', false)
print(ESX.DumpTable(item))
```
</CodeGroup>

<Columns cols={2}>
  <div>
    ### Parameters

    ข้อมูลที่ต้องส่งไปยังฟังก์ชั่น

    <ParamField path="name" type="string" required>
      ชื่อไอเทม
    </ParamField>
    <ParamField path="validate" type="boolean" default={false}>
      ตรวจสอบบน Server หรือไม่ ถ้าหากไม่กำหนดหรือ false จะตรวจสอบเฉพาะ Client
    </ParamField>
  </div>
  <div>
    ### Return

    สำคัญมีโอกาสเป็น `nil`

    <ResponseField name="name" type="string" required>ชื่อไอเทม</ResponseField>
    <ResponseField name="label" type="string" required>คำแสดงทั่วไป</ResponseField>
    <ResponseField name="count" type="integer" required>จำนวนที่มี</ResponseField>
    <ResponseField name="rare" type="boolean" required>ไอเทมชิ้นนี้พิเศษหรือไม่</ResponseField>
    <ResponseField name="canRemove" type="boolean" required>สามารถลบออกจากกระเป๋าได้หรือไม่</ResponseField>
    <ResponseField name="limit" type="integer" default={-1}>จำนวนที่จำกัดในการเก็บ</ResponseField>
    <ResponseField name="weight" type="number" default={0}>น้ำหนักต่อชิ้น</ResponseField>
  </div>
</Columns>

## Information - ข้อมูลเบื้องต้น

- **เวอร์ชั่นปัจจุบัน** `1.0.0`

## Requirements - ความต้องการขั้นพื้นฐาน

- **FXServer** (ต้องรองรับ Onesync, GlobalState)
- **ESX**
- **ox_lib**

## Installation - ติดตั้ง

[Download](https://github.com/mysbryce/ns_core)

ให้คุณไปที่ไฟล์ `fxmanifest.lua` ของโฟลเดอร์ **Project** ที่คุณต้องการใช้งาน `ns_core` และเพิ่ม

```lua
shared_scripts {
	'@ox_lib/init.lua',
}

client_scripts {
  '@ns_core/lib/client.lua'
}

server_scripts {
  '@ns_core/lib/server.lua'
}
```

## State

สร้างตัวแปรพร้อมกับการ **Handle Value Change** ซึ่งจะทำให้คุณสามารถตรวจสอบควบคุมการเปลี่ยนแปลงตัวแปรได้ในทันทีเมื่อมีการเปลี่ยน

โดยการใช้งานมีดังนี้

```lua
-- NS.createState(value, fn)

local hasOpen, setHasOpen = NS.createState(true, function(previous, current)
	print('log: previous', previous)
	print('log: current', current)
end)

print('hasOpen', hasOpen)
setHasOpen(false)

-- output:
-- hasOpen true
-- log: previous true
-- log: current false
```

| Parameter | Type                             | Description                          | Note |
| --------- | -------------------------------- | ------------------------------------ | ---- |
| value     | any                              | ค่าเริ่มต้น                          |      |
| fn        | fun(previous: any, current: any) | ฟังก์ชั่นเมื่อมีการเปลี่ยนแปลงข้อมูล |      |
